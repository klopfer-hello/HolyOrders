# HolyOrders

![HolyOrders](media/banner.png)

Paladin blessing coordination for WoW Classic (TBC Anniversary) — assign,
sync, and cast raid blessings without wasting raid time.

A fully independent, from-scratch implementation. Key ideas:

- **Remembers your plans.** Blessing assignments are stored per raid roster and
  re-applied automatically when you raid with the same people again.
- **Solo-paladin mode.** Alone as a paladin? Salvation goes on everyone except
  tanks, automatically.
- **Reliable sync.** The comm protocol is designed so what you see is what
  every paladin sees — assignments don't silently revert or get lost.
- **Deterministic auto-assign.** Same raid, same talents → same plan, every time.
- **Pets included.** Hunter and warlock pets are tracked and buffed properly.

## Status

Early development — not yet playable.
